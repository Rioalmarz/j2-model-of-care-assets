## Usage

To use these images in your project, simply copy the raw link and use it in your HTML, Markdown, or other content:

```markdown
![Diabetes](https://raw.githubusercontent.com/Rioalmarz/j2-model-of-care-assets/main/Diabetes.jpg)
```

```html
<img src="https://raw.githubusercontent.com/Rioalmarz/j2-model-of-care-assets/main/Diabetes.jpg" alt="Diabetes">
```
